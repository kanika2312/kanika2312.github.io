My Design Process

"There’s a big difference between making a simple product and making a product simple" — Des Traynor

Every project is different, so I employ an iterative and adaptable design process. My design process typically starts with understanding the user needs and brainstorming. I then use pen and paper to sketch out rough ideas, create user flows to complete each goal in least number of steps. I then create wireframes using Sketch and transform those into high fidelity mockups by experimenting with different colours, styles and typography that best suits the needs. I then use invision to build interactive prototypes. Finally conduct user testing and iterate the above steps to make changes as per the user goals.
